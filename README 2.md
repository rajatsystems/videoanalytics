# YOLO Streamlit Dashbord
Display predicted Video, Images and webcam using YOLO models (**YOLOv7** & **YOLOv8**) with Streamlit


## Streamlit Options
### Modes
 - RTSP
 - Webcam
 - Video
 - Image

 ## StepUp
```
git clone project (github project link)
cd streamlit-yolov7
```
Install dependency
```
pip3 install -r requirements.txt
```
Run **Streamlit**
```
streamlit run app.py
```
